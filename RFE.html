How to use this tool:

  1. Find your Rust config folder. It's usually here:
    /steamapps/common/Rust/cfg/
  2. Drag the favorites.cfg file from that folder into the box above, or click the button to select it.
  3. Edit, add, or delete your servers using the tools below.
  4. Click the "Download favorites.cfg" button to save your changes.
  5. Replace the old favorites.cfg file in the Rust folder with your new downloaded one.
